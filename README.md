# Performance-Intelligence-App

A personal performance intelligence app designed to work alongside Zepp and an Amazfit tracker.

Core philosophy:

> Zepp records what happened. This app explains whether I am becoming a better athlete.

The app focuses on athletic performance trends across running, strength training, basketball, recovery, training load, personal records, and weekly reviews.
